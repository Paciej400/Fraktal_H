Po uruchomieniu programu pojawi się okno z polem do wpisania ilości rekurencji (poziomów), z jakimi fraktal oparty na literze "H" zostanie wygenerowany. Po wpisaniu wcisnąć ENTER. Dla liczb większych od 5 nie widać różnicy. Można wpisać 0 aby wyczyścić ekran. 

Zalecana wersja Javy: 8 - 10 (program korzysta z biblioteki JavaFX).

Środowisko, w którym tworzyłem program: IntelliJ.
